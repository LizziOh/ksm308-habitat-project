
# ksm308-habitat-project

<!-- badges: start -->
<!-- badges: end -->

This repository provides teaching resources to run the subtidal habitat project for the marine science unit KSM308, project which allows students to collect marine imagery, assess imagery and analyse data and report on the differences between subtidal habitat composition at different sites and likely environmental and biological drivers.  Included are: the scheme used to annotate the collected images with reference images, a workflow to organise images and metadata for import into Squidle+ (the image annotation platform) and code to export data from Squidle+ via the API


*Output contains example exported data*

*Resources contains the label scheme and reference image catalogue*

*Scripts contains code to help import image catalogues into SQuidle and export data*
